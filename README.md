Проект включает в себя SQL-аналитику поведения пользователей и анализ эффективности A/B теста в игровом магазине


Содержание

1. SQL

Решение аналитических задач на MySQL:

Расчет выручки конкретных когорт платежей

Метрики вовлеченности (Login/Payment frequency)

Анализ и прогнозирование LTV (7/30 дней)



2. A/B тест

Анализ изменения ценообразования стартового набора (снижение цены в 5 раз, увеличение наполнения x2)

Стек: Python, Pandas, SciPy, Seaborn

Результаты:

CR (Конверсия) в тестовой группе выросла

ARPPU значительно снизился из-за падения цены

ARPU (общая выручка) показал негативную динамику в моменте

Вывод: Рекомендовано не делать обновление без анализа долгосрочного Retention, так как текущий рост конверсии не перекрывает падение среднего чека



Как запустить

1. SQL запросы находятся в файле "sql/task1_queries.sql"

2. Python ноутбук с графиками — "notebooks/ab_test_analysis.ipynb"
